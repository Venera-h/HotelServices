# Сценарии тестирования Saga Choreography

## Сценарий 1 — Успешное бронирование
- Заказ: ORD-001, отель: H1, сумма: 300
- H1 имеет свободные номера → номер забронирован
- Сумма 300 <= лимита 500 → оплата прошла
- Итог: заказ CONFIRMED ✅

## Сценарий 2 — Нет свободных номеров
- Заказ: ORD-002, отель: H2, сумма: 200
- H2 не имеет свободных номеров → бронирование отклонено
- До оплаты дело не доходит
- Итог: заказ CANCELLED ❌

## Сценарий 3 — Оплата отклонена (компенсация)
- Заказ: ORD-003, отель: H1, сумма: 999
- H1 имеет свободные номера → номер забронирован
- Сумма 999 > лимита 500 → оплата отклонена
- Компенсация: HotelService автоматически освобождает номер
- Итог: заказ CANCELLED ❌, номер возвращён в инвентарь 🔄
